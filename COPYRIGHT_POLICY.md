## 1. 当社の基本方針

**Persona AI**は、知的財産権を尊重し、著作権侵害に対して迅速かつ適切に対応します。

### 1.1 法的根拠

当社は、以下の法律に準拠して著作権侵害に対応します：

- 🇯🇵 **日本**: 著作権法（昭和45年法律第48号）、プロバイダ責任制限法（平成13年法律第137号）
- 🇺🇸 **米国**: デジタルミレニアム著作権法（DMCA, 17 U.S.C. § 512）

### 1.2 当社の立場

当社は**プラットフォーム提供者**であり、ユーザーが投稿したコンテンツの著作権侵害について、以下の条件下で免責されます：

✅ **免責要件**:
- 侵害を知らなかった（事前認識がない）
- 通知受領後、速やかに削除した
- 技術的に可能な範囲で侵害防止措置を講じていた

## 2. 著作権侵害の定義

以下の行為は、著作権侵害にあたる可能性があります：

### 2.1 禁止される行為

❌ **キャラクター名の無断使用**
```
例:
- ドラえもん
- ミッキーマウス
- ポケモン（ピカチュウ等）
- 呪術廻戦のキャラクター
- 推しの子のキャラクター
```

❌ **著名人の名前・肖像の無断使用**
```
例:
- 芸能人の名前を使った人格AI
- 声優・VTuberの名前を使った人格AI
- スポーツ選手の名前を使った人格AI
```

❌ **他者の作品の無断転載**
```
例:
- 小説・漫画の文章をそのまま使用
- イラスト・画像の無断転載
- 楽曲の歌詞を無断で使用
```

❌ **商標・ロゴの不正利用**
```
例:
- 企業ロゴを無断で使用
- 商品名・サービス名を不正に使用
- ブランド名を偽装する行為
```

### 2.2 許可される行為

✅ **適法な使用**
```
以下の場合は、著作権侵害にあたらない可能性があります：

1. パロディ・風刺（日本法では限定的）
2. 引用（出典を明記し、必要最小限の範囲）
3. 権利者の明示的な許諾を得た場合
4. 著作権保護期間が切れた作品（パブリックドメイン）
5. クリエイティブ・コモンズ（CC）ライセンスに準拠した使用
```

⚠️ **注意**: 日本では「パロディ」が米国ほど広く認められていないため、慎重に判断してください。

## 4. 削除プロセス

当社は、以下のプロセスで削除対応を行います。

### 4.1 対応フロー

```
1. 削除要請受領
   ↓ (1時間以内)
2. 初期確認（要件充足チェック）
   ↓ (6時間以内)
3. 侵害内容の審査
   ↓ (24時間以内)
4. 削除実施 or 却下
   ↓
5. 両者への通知
   ↓ (14日間)
6. 異議申し立て期間
   ↓
7. 最終判断
```

### 4.2 対応期限

| 緊急度 | 判断基準 | 対応期限 | 対応内容 |
|--------|---------|---------|---------|
| **緊急** | 大企業・公的機関からの申立 | 24時間以内 | 即座に削除・アカウント停止 |
| **通常** | 一般的な著作権侵害 | 72時間以内 | 確認後、削除措置 |
| **軽微** | 軽微な侵害の疑い | 7営業日以内 | 警告・改善要求 |

### 4.3 削除後の措置

削除後、当社は以下の措置を講じます：

- ✅ コンテンツの即座の削除
- ✅ ユーザーへの通知メール送信
- ✅ 著作権者への完了報告
- ✅ ログの保存（6ヶ月間）

## 5. 異議申し立て

削除されたユーザーは、以下の要件を満たす場合、異議申し立てができます。

### 5.1 異議申し立ての要件

✅ **認められる理由**:
- 適法な使用（パロディ、引用等）
- 権利者の許諾を取得済み
- 誤認（著作物の類似性がない）
- 著作権保護期間が切れている

### 5.2 異議申し立ての方法

**📧 異議申し立て専用窓口**
- **Email**: appeals@persona-ai.com
- **件名**: 「削除に対する異議申し立て」
- **期限**: 削除通知受領後14日以内

### 5.3 必要書類

```
【必須項目】
1. ユーザー情報
   - ユーザーID:
   - Email:

2. 削除されたコンテンツ
   - 人格AI ID:
   - 削除日時:

3. 異議の理由
   （以下から選択）
   - [ ] 権利者の許諾を取得済み
   - [ ] 適法な使用（パロディ、引用等）
   - [ ] 誤認（類似性がない）
   - [ ] その他: _______________

4. 証拠書類
   - 許諾書（権利者の署名・捺印付き）
   - 契約書
   - その他の証明書類

5. 宣誓
   「私は、上記の情報が正確であり、適法な使用であることを
    誠実に信じていることを宣誓します。」

   署名: _________________
   日付: _________________
```

### 5.4 再審査プロセス

```
1. 異議申し立て受領
   ↓ (3営業日以内)
2. 証拠書類の確認
   ↓ (7営業日以内)
3. 著作権者への通知・意見聴取
   ↓ (7営業日以内)
4. 最終判断
   ↓
5. 復元 or 却下
```

## 6. 反復侵害者への対応

当社は、著作権侵害を繰り返すユーザーに対して、厳格な措置を講じます。

### 6.1 3ストライクポリシー

```
【1回目】⚠️ 警告
- 警告メール送信
- 人格AIの削除
- 監視対象に追加

【2回目】🚫 アカウント一時停止
- アカウント停止（7日間）
- 全人格AIの非公開化
- 最終警告メール送信

【3回目】❌ アカウント永久停止
- アカウント永久削除
- 全データの削除
- IPアドレスのブロック
- 再登録の禁止
```

### 6.2 悪質な場合の対応

以下の場合は、1回でアカウント永久停止とします：

- 大量の著作権侵害を意図的に行った場合
- 商業目的での侵害
- 削除後に同一コンテンツを再投稿した場合
- 虚偽の異議申し立てを行った場合

## ✅ よくある質問（FAQ）

### Q1: パロディは許可されますか？

**A**: 日本では、パロディが著作権侵害にあたる可能性があります。米国と異なり、日本ではパロディの保護が限定的です。著作権者の許諾を得ることを強く推奨します。

### Q2: 削除要請はどのくらいで対応されますか？

**A**: 通常72時間以内に対応します。緊急の場合は24時間以内に対応します。

### Q3: 異議申し立てはできますか？

**A**: はい。削除通知受領後14日以内に、証拠書類を添えて異議申し立てができます。

### Q4: 公式バッジはどうすれば取得できますか？

**A**: 管理者に申請してください。著作権者であることの証明が必要です。
