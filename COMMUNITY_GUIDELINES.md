## 1. はじめに

**Persona AI**は、クリエイターとユーザーが安全に交流できるコミュニティを目指しています。

本ガイドラインは、すべてのユーザーが快適に本サービスを利用するためのルールです。

---

## 2. 基本原則

### 2.1 尊重

- 他のユーザーを尊重する
- 多様な意見・価値観を認める
- 丁寧なコミュニケーションを心がける

### 2.2 安全性

- 自分と他者の安全を最優先する
- 個人情報を守る
- 不適切なコンテンツを投稿しない

### 2.3 誠実さ

- 嘘や誤情報を広めない
- なりすましをしない
- 著作権を尊重する

---

## 3. 禁止事項

以下の行為は禁止します。違反した場合、アカウント停止等の措置を講じます。

### 3.1 暴力的・差別的なコンテンツ

❌ **禁止例**:
```
- 暴力の脅迫・煽動
- 人種・性別・宗教等に基づく差別
- ヘイトスピーチ
- 自傷・自殺の煽動
```

### 3.2 わいせつ・ポルノグラフィックなコンテンツ

❌ **禁止例**:
```
- わいせつな画像・動画
- 性的な内容の人格AI
- 未成年を性的対象とする内容
- リベンジポルノ
```

### 3.3 ハラスメント・誹謗中傷

❌ **禁止例**:
```
- 個人への嫌がらせ
- 誹謗中傷・名誉毀損
- ストーキング行為
- ドクシング（個人情報の暴露）
```

### 3.4 スパム・広告

❌ **禁止例**:
```
- スパム投稿
- 商業目的の宣伝（許可なし）
- フィッシング詐欺
- マルウェアの配布
```

### 3.5 知的財産権の侵害

❌ **禁止例**:
```
- 有名キャラクターの無断使用
- 著名人の名前・肖像の無断使用
- 他者の作品の無断転載
- 商標・ロゴの不正利用
```

詳細は、[著作権ポリシー](./COPYRIGHT_POLICY.md)をご確認ください。

### 3.6 不正アクセス・システム妨害

❌ **禁止例**:
```
- ハッキング・クラッキング
- DDoS攻撃
- 自動化ツールの悪用
- 脆弱性の悪用
```

## 4. 推奨行動

以下の行動を推奨します。

### 4.1 クリエイティブな人格AI

✅ **推奨例**:
```
- オリジナルキャラクターの作成
- 架空の設定・世界観の構築
- ユーザーに楽しんでもらえる工夫
- 独自の話し方・個性の設定
```

### 4.2 建設的なフィードバック

✅ **推奨例**:
```
- 丁寧な評価コメント
- 改善提案
- バグ報告
- 他のクリエイターへの応援
```

### 4.3 適切な通報

✅ **推奨例**:
```
- 不適切なコンテンツを発見したら通報
- 具体的な理由を記載
- 虚偽の通報はしない
```

---

## 5. 違反時の対応

違反行為に対して、以下の措置を講じます。

### 5.1 段階的措置

```
【1回目】⚠️ 警告
- 警告メール送信
- 違反コンテンツの削除

【2回目】🚫 アカウント一時停止
- 7日間のアカウント停止
- 再発防止の誓約書提出

【3回目】❌ アカウント永久停止
- アカウント永久削除
- IPアドレスのブロック
```

### 5.2 即座の永久停止

以下の場合は、1回でアカウント永久停止とします：

- 児童ポルノ
- テロの煽動
- 重大な著作権侵害（商業目的）
- 重大なハラスメント

## 6. 通報システム

不適切なコンテンツを発見した場合は、通報してください。

### 6.1 通報方法

**Email**: report@persona-ai.com  
**件名**: 「コンテンツの通報」

### 6.2 必要情報

```
【必須項目】
1. 通報するコンテンツのURL
2. 通報理由（以下から選択）
   - [ ] 暴力的・差別的
   - [ ] わいせつ・ポルノグラフィック
   - [ ] ハラスメント・誹謗中傷
   - [ ] スパム・広告
   - [ ] 著作権侵害
   - [ ] その他: _______________

3. 詳細説明（具体的に記載）

4. スクリーンショット（あれば）
```

### 6.3 対応期限

- 通常: 72時間以内に確認・対応
- 緊急: 24時間以内に対応
