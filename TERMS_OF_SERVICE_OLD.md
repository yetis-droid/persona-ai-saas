**以下の行為を禁止します。違反した場合、即座にアカウント停止および法的措置を講じます。**

❌ **禁止例**:
```
- 有名キャラクター名の無断使用（例: ドラえもん、ポケモン、ミッキーマウス）

- 他者の作品・画像の無断転載
- 商標・ロゴの不正利用
```

### 5.2 なりすまし

❌ **禁止例**:
```
- 「【公式】」「本物」等の虚偽表示

- 公式アカウントを偽装する行為
```

### 5.3 不適切なコンテンツ

❌ **禁止例**:
```
- 暴力的・差別的な内容
- わいせつ・ポルノグラフィックな内容
- 他者を誹謗中傷する内容
- スパム・広告目的の投稿
```

### 5.4 不正アクセス・システム妨害

❌ **禁止例**:
```
- ハッキング・クラッキング行為
```

## 7. 著作権侵害への対応

**当社は知的財産権を尊重し、著作権侵害に迅速に対応します。**

### 7.1 DMCA準拠の削除プロセス

当社は、デジタルミレニアム著作権法（DMCA）および日本の著作権法に準拠して、著作権侵害の申し立てに対応します。

### 7.2 削除要請の方法

著作権者は、以下の連絡先から侵害コンテンツの削除を要請できます：

**📧 著作権侵害専用窓口**
- **Email**: copyright@persona-ai.com
- **件名**: 「著作権侵害の申し立て」
- **対応時間**: 24時間以内に対応

**必要情報**:
```
1. 著作権者の氏名・連絡先
2. 侵害されている著作物の詳細（タイトル、URL等）
3. 侵害コンテンツのURL・ID
4. 著作権者であることの証明（登記簿、作品掲載ページ等）
5. 誠実な申立であることの宣誓
```

### 7.3 対応期限

| 緊急度 | 対応期限 | 対応内容 |
|--------|---------|---------|
| **緊急** | 24時間以内 | 即座に削除・アカウント停止 |
| **通常** | 72時間以内 | 確認後、削除措置 |
| **軽微** | 7営業日以内 | 警告・改善要求 |

### 7.4 異議申し立て

削除されたユーザーは、以下の要件を満たす場合、異議申し立てができます：

- 適法な使用（パロディ、引用等）
- 権利者の許諾を取得済み
- 誤認（著作物の類似性がない）

**手続き**:
- Email: appeals@persona-ai.com
- 期限: 削除通知受領後14日以内
- 証拠: 許諾書、契約書等の添付が必要

### 7.5 反復侵害者への対応

**3ストライクポリシー**:
```
1回目: 警告メール送信
2回目: アカウント一時停止（7日間）
3回目: アカウント永久停止
```

## 9. 免責事項と責任制限

### 9.1 プラットフォーム提供者の免責

当社は「プラットフォーム提供者」であり、ユーザーが投稿したコンテンツについて、以下の範囲で免責されます：

✅ **免責要件**:
- 侵害を知らなかった場合
- 通知受領後、速やかに削除した場合
- 技術的に可能な範囲で侵害防止措置を講じていた場合

**法的根拠**: プロバイダ責任制限法（平成13年法律第137号）第3条

### 9.2 ユーザーの責任

著作権侵害を行ったユーザーは、以下の責任を負います：

- **民事責任**: 著作権者への損害賠償
- **刑事責任**: 著作権法違反（10年以下の懲役または1,000万円以下の罰金）
- **当社への賠償**: 当社が被った損害の賠償

### 9.3 責任の制限

当社に故意・重過失がある場合を除き、以下について責任を負いません：

- ユーザー間のトラブル
- サービスの中断・停止による損害
- データの消失・破損
- 第三者によるハッキング等
